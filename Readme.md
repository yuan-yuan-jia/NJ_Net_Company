### 目的
关于互联网公司的帖子都是北上广深杭的，南京的介绍比较少，作为外包之都，很多人找工作都是属于开盲盒，所以想整理一份关于南京互联网公司情况的表格。如果是使用腾讯文档，容易被封禁，无法访问，使用 google doc 有时候就会出现有人随意填写，破坏格式的情况，所以我想通过 issue 结合 github action 来完成这项工作。

### 功能
1. 填写issue自动生成表格到当前仓库
2. 结果同步google doc
3. 结果通过telegram bot同步到telegram群组，bot名称 @NJ_Net_Company_bot, 群组名称 [在南京的程序员](https://t.me/+Kwm8tW1UxSZlMzll)

### 如何使用

1. 进入仓库，点击 issue ，进入 issue 列表页面，可以先搜索一下是否有自己的公司，如果有，则直接进入该 issue 评论添加情况即可，如没有，则 new issue
2. 进入 issue 编辑页面，标题最好填写公司名称+部门名称，部门也可不填写
3. issue 模板是 json 格式的，只需要填写对应的 value 即可，编辑完成后，直接提交，github ci 会自动生成对应的 markdown 和 excel，最新结果可在本仓库*南京互联网公司统计汇总.md*和*南京互联网公司统计汇总.xlsx*中查看
